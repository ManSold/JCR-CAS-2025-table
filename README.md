# 2025 期刊分区表（本地工作流用）

本目录用于文献检索工作流中的期刊质量分级。

## 文件

- `CAS_2025.csv`：2025 中科院分区表，字段包括：
  - 期刊名称
  - 2025分区
  - Top
  - Open Access

- `JCR_2025.csv`：2025 JCR 分区表，字段包括：
  - 期刊名
  - ISSN
  - eISSN
  - Category
  - Total citation
  - 2024JIF
  - Quartile
  - JIF rank
  - 2023分区
